# 미완성한 기능들
1. 재고 관리 화면에 현재 재고를 표시하는 기능과 + - 버튼을 통해 재고를 관리하는 기능
2. 재고를 수정하거나 메뉴를 주문했을 때 변경된 재고의 값을 라벨에 업데이트하는 기능
3. 재고 관리 화면을 닫는 버튼 미완성
